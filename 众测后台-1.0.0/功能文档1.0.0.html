一 众测商品管理

  4. 编辑在线商品：线上的众测商品如果有内容上的错误，可以在后台进行编辑，但可编辑的部分仅限于商品名、分类、封面图、商品描述，不能编辑涉及到活动规则的内容。

二 商品管理流程

三 申请众测

众测商品包含两个时间段：
  1. 申请期：在申请期，用户参与活动的报名。
  2. 体验期：在体验期，成功申请的用户要在周期结束之前提交众测文章。
处理用户申请
  • 申请期结束后，需要在后台处理所有的用户申请。
  • 按照对应的商品数量，选择数量一致的申请用户作为成功申请的用户。
  • 对应每一个成功申请的用户，都会生成相应的订单。
  • 补充每一个订单上的物流信息，整个申请处理环节才算完成。
违规记录
  • 当成功申请的用户没有在规定的时间提交测评文章将被系统记录为一次违规。
  • 在审核活动名单的时候，用户的违规记录将作为一个重要的参考，影响用户最终是否能通过审核。
